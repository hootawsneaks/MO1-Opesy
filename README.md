# osqts command line emulator
made by kean carvin, gideon chua, daniel pua, and bea uy

## requirements:
- Visual Studio 2022 with the "Desktop development with C++" workload (this gives MSVC + CMake)
- or VS Build Tools + CMake separately

## installation and run flow:

```
git clone <repo>
cd "MO1 Opesy"
cmake -S . -B out/build/x64-debug
cmake --build out/build/x64-debug
```
